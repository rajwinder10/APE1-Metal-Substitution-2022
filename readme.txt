#  APE1-Metal-Substitution-2022

The model naming is as follows: The type of metal (Mg, Mn, Ni, Zn, Ca) followed by the QM/MM-optimized stationary point (RC, IC, PC, TS1, TS2).
"Ni-oct" refers to Ni in octahedral coordination geometry in the APE1 active site
"Ni-sp" refers to Ni in square-planar coordination geometry in the APE1 active site
